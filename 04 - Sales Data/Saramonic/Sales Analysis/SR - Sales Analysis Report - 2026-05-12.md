## 6. Caveats & Data Quality Notes

- **File `Sales by SKU (QTY).md` saat ini berisi VALUE data** (duplikat dari Value file). Forecast SKU dilakukan berbasis VALUE (Rp), bukan unit QTY. Perlu fix file QTY untuk analisis unit-based.
- Mar 2026: ada inkonsistensi kecil antara file Categorized (Rp 1,055,174,710) vs SKU (Rp 1,014,226,710). Selisih ~41M (4%). Pakai versi SKU untuk konsistensi total brand.
- **May 2026 partial** — data cut-off di awal bulan, total Rp 91.8M. Forecast May di file XLSX sudah memperhitungkan partial run-rate ini.
- Data **sell-IN** (Saramonic → dealer). Sell-out ke consumer mungkin beda — cross-check dengan Claim Cashback & Credit Note Usage files untuk full picture.
- "Dormant" definisi: 2+ bulan consecutive zero atau <10% dari average historis.
